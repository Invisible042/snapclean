# SnapClean

SnapClean is a background remover app that allows users to upload images, remove their backgrounds, and preview the results in real time. This project is currently in development and is part of my learning journey in full-stack development.

---

## Project Goals

- Build a clean and responsive UI using React and Tailwind CSS.
- Implement image processing and background removal features.
- Use Cloudinary for image storage and optimization.
- Integrate a Node.js and Express.js backend.

---

## Current Progress

- ✅ Basic UI layout with upload and slider functionality.
- ✅ Real-time preview using sample images.
- 🚧 Backend integration with Cloudinary (in progress).
- 🚧 Add the ability to download processed images.

---

## Upcoming Features

- User authentication to save and manage processed images.
- Optimize the background removal process for better performance.
- Build a portfolio-ready demo site.
